Commercial Support

ASF support

Please note that the Apache Software Foundation does not offer commercial support for Apache Grails or related applications. If you encounter issues or wish to propose changes, you can submit a change request and contribute code to the project. Ensure you meet the requirements specified here.

List of companies

How to add a new company

You can create a Pull Request with the required information. A committer will review it, ensuring accuracy, or seek clarification. Alternatively, you can submit a request to the Dev Mailing list.
